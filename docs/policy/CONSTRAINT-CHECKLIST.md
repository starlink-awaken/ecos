## 使用方式

设计前, 逐项确认。发现"不支持"的项 → 在设计中标为 Phase N。

例如:
  Phase 1 设计8角色委员会前:
    delegate_task最大并发: 3 → 结论: Phase 1不可行
    设计修正: 三角模式(3角色并发)
  
  Phase 3 设计实时hook前:
    操作前hook: 否 → 结论: 当前不可真实时
    设计修正: 脚本检查 + Agent自觉
