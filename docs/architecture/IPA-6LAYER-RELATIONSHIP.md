# IPA 三层模型与六层架构的关系

> 统一 GENOME.md 中的两套模型
> 版本: v0.1.0
> 创建: 2026-05-13（逻辑推演后补文档）

---

## 两套模型的本源

**IPA 三层**（GENOME.md 第五部分）：
```
A — Action（行动层）
I — Intelligence（智能层）
P — Persistence（持久层）
```

**六层架构**（GENOME.md 第六部分）：
```
L6: 反馈层
L5: 行动层
L4: 智能层
L3: 持久层
L2: 感知层
L1: 宪法层
```

---

## 两套模型的定位

| 模型 | 定位 | 视角 | 用途 |
|------|------|------|------|
| IPA | **运行时模型** | 认知操作流 | 描述一次认知循环中信息如何流动 |
| 六层 | **部署/架构模型** | 组件分层 | 描述系统有哪些组件，组件间依赖关系 |

**两者不是竞争关系，是互补关系。**

---

## 映射关系

```
        六层架构                          IPA (运行时视角)
    ┌──────────────┐                    ┌──────────────┐
    │ L6 反馈层     │ ←── 稳态监控 ──→   │              │
    ├──────────────┤                    │  反馈回路    │
    │ L5 行动层     │ ←───────────────── │  A (Action)  │
    ├──────────────┤                    ├──────────────┤
    │ L4 智能层     │ ←───────────────── │  I (Intelligence)
    ├──────────────┤                    ├──────────────┤
    │ L3 持久层     │ ←───────────────── │  P (Persistence)
    ├──────────────┤                    └──────────────┘
    │ L2 感知层     │ ←── 五阶管道 ──→   输入（IPA 未显式建模）
    ├──────────────┤
    │ L1 宪法层     │ ←── 不变量 ───→   约束（IPA 未显式建模）
    └──────────────┘
```

**IPA 缺失的层：**
- 感知层 → IPA 假定输入已经是结构化的。但六层明确要求五阶降熵管道。
- 宪法层 → IPA 不包含自我约束。六层显式建模 L0/L1/L2 不变量。
- 反馈层 → IPA 的贝叶斯更新是单向的（A→P→I），缺少负反馈回路。

## 统一模型：eCOS 完整运行时

```
                    ┌─────────────── L1 宪法约束 ───────────────┐
                    ↓                                            ↓
  外界信息 ──→ L2 感知(五阶降熵) ──→ L3 持久(P) ──→ L4 智能(I) ──→ L5 行动(A)
                    ↑                    ↑              │            │
                    │                    │              │            │
                    │                    └── 知识更新 ──┘            │
                    │                                                │
                    └──── L6 反馈(三类稳态检查) ←── 行动结果评估 ←───┘
                                   │
                                   ├── 宪法偏离？ → 停止→通知人类
                                   ├── 知识偏离？ → 重新感知
                                   └── 目标偏离？ → 重新委员会
```

**这个统一模型明确了：**
1. **感知是 IPA 的输入预处理**，不能跳过
2. **宪法在所有层之上约束**
3. **反馈形成闭环**，而非 IPA 的单向链
4. **SSB（用虚线箭头表示）贯穿所有层，传输信息**

## 贝叶斯更新在统一模型中的位置

```
统一模型中的贝叶斯回路：

先验信念(持久层L3) → 似然度评估(智能层L4) → 行动测试(行动层L5)
                                         ↓
                    后验信念更新(持久层L3) ← 结果评估(反馈层L6)
```

这是一个完整的贝叶斯认知循环，而非单向流。
